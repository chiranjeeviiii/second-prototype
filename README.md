# RCM Poster Generator

A simple web app to generate congratulation posters for RCM achievements.

Users can upload a photo, enter their name, select an achievement, and download a ready-made poster.

---

## Features

User side:

• Upload photo  
• Enter name  
• Select achievement  
• Generate poster  
• Download poster  

Admin side:

• Admin login  
• Template configuration generator  
• Helps create new frame JSON entries
